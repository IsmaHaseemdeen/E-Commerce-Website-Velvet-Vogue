# E-Commerce-Website-Velvet-Vogue
E-Commerce Website: Velvet Vogue
Velvet Vogue is a comprehensive e-commerce platform developed to offer trendy casualwear and sophisticated formal wear, alongside accessories, targeting young adults. The site is built using HTML, CSS, and JavaScript for a responsive frontend, with PHP managing server-side logic and SSMS serving as the database. Key features include detailed product catalogues (Men's, Women's, Kid's, Accessories) with "Add to Cart" and "Buy Now" options, dedicated sections for promotions, and essential forms for user registration (Sign-up) and processing transactions (Place Order)
